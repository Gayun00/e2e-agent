# 제품 개요

Playwright E2E Agent는 자연어 설명을 사용하여 end-to-end 테스트를 생성하는 AI 기반 테스트 생성기입니다. Page Object Model (POM) 패턴을 따르며 Anthropic의 Claude API와 통합됩니다.

## 핵심 기능

- 자연어를 Playwright 테스트로 변환
- Page Object Model 패턴 구현
- 자동 요소 선택자 감지
- 모킹을 지원하는 스크린샷 생성
- 자동 로그인 감지 및 처리
- 테스트 컨텍스트를 위한 도메인 지식 관리

## 대상 사용자

POM 모범 사례를 따르면서 깔끔하고 유지보수 가능한 테스트 코드를 유지하며 AI 지원을 통해 E2E 테스트 생성을 가속화하려는 개발자 및 QA 엔지니어.
